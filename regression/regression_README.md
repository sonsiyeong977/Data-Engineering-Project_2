## 🔬 분석 개요

### 회귀 모델

$$
y = b_0 + b_1Q + b_2E + b_3S + 1.5 \cdot b_4P
$$

| 변수 | 설명 |
|------|------|
| $y$ | 판매량 역산 지표 (올리브영 리뷰 수) |
| $Q$ | 품질 지수 (리뷰 기반 품질·효능 감성) |
| $E$ | 사회적 가치 지수 (입소문·SNS 반응) |
| $S$ | 감성 지수 (디자인·향 등 감성 만족) |
| $P$ | 가격 지수 (ml당 가격 기반 가성비) |

> 가성비 중심 추천 시스템 구축을 위해 $P$에 **1.5배 가중치** 선반영

## 📊 분석 1 — 일반 상품 (378개)

### 데이터

| 항목 | 값 |
|------|----|
| 분석 대상 | 5개 카테고리 일반 상품 |
| 유효 데이터 | 378개 |
| 종속변수 | `olive_review_count` (리뷰 수) |
| 독립변수 | Q, E, S, P 점수 |

### 회귀 계수 (b값)

| 변수 | 계수 (b) | 해석 |
|------|---------|------|
| 상수 ($b_0$) | 2,203.93 | 기본 판매 지수 |
| 품질 ($b_1 \cdot Q$) | **7,765.83** | 판매량에 강력한 정(+) 영향 |
| 가치 ($b_2 \cdot E$) | 885.02 | 상대적으로 낮은 영향력 |
| 감성 ($b_3 \cdot S$) | -18,086.09 | 통계적 노이즈 → 보정 적용 |
| 가성비 ($b_4 \cdot P$) | **8,218.74** | **최고 영향력 확인** |

> S(감성) 계수 음수는 통계적 노이즈로 판단 → 추천 엔진 적용 시 양수 보정

### 최종 가중치 (보정 후 정규화)

| 추천 요소 | 최종 가중치 |
|-----------|------------|
| **가성비 (P)** | **47.32%** |
| **품질 (Q)** | **44.71%** |
| 사회적 가치 (E) | 5.10% |
| 감성 (S) | 2.87% |

> **가성비 + 품질 합산 약 91%** → 제품 인기를 결정짓는 핵심 요소로 확인

## 📊 분석 2 — 프로모션 상품 (166개)

### 데이터

| 항목 | 값 |
|------|----|
| 분석 대상 | `olive_is_promo == True` 프로모션 상품 |
| 유효 데이터 | 166개 |
| 추가 변수 | `avg_review_score` (평균 별점) |

### 회귀 계수 (b값)

| 변수 | 계수 (b) | 해석 |
|------|---------|------|
| 품질 ($Q$) | -36,225 | 음수 → 프로모션 시장에서 외면 |
| 가치 ($E$) | **+14,199** | 강력한 구매 동인 |
| 감성 ($S$) | -20,617 | 음수 → 프로모션 시장에서 외면 |
| 가성비 ($P$) | **+25,960** | **가장 압도적인 구매 동인** |

### 핵심 인사이트

**구매 동인 (Positive)**
- **가성비(P)**: 프로모션 상품 구매 시 경제적 혜택을 가장 예민하게 따짐
- **가치(E)**: "필요한 효능 제품이 세일할 때" 지갑을 여는 패턴 확인

**배제 요인 (Negative)**
- **품질(Q)**: 다기능·프리미엄 제품은 프로모션보다 정가 구매 선호
- **감성(S)**: 화려한 패키징 제품은 프로모션 대량 구매 기피

> **비즈니스 인사이트:** 기획전 최상단에는 데일리로 사용하는 에센스·토너류
> 고효능(E) 1+1/증량 상품(P)을 배치해야 전환율(CVR) 극대화 가능

### 최종 추천 로직

$$
Final\_Score = 0.8 \times Value\_Score + 0.2 \times Promo\_Bonus
$$

> Top 5 결과: **에센스/세럼/앰플 카테고리 압도적 강세**
> → "가치(E)에 충실하면서 가성비(P)가 좋은" 조건에 에센스 기획 세트가 최적 부합

## 🔗 추천 시스템 연결

### 사용자 맞춤 Value Score 계산식

$$
User\_Value\_Score = (b_1 \cdot w_Q)Q + (b_2 \cdot w_E)E + (b_3 \cdot w_S)S + (b_4 \cdot w_P \cdot 1.5)P
$$

| 구성 요소 | 역할 |
|-----------|------|
| 기본 계수 ($b_n$) | 실제 판매 데이터 기반 영향력 (회귀분석 결과) |
| 사용자 가중치 ($w_n$) | 사용자 개인 선호도 (슬라이더 입력) |
| 가성비 상수 (1.5) | 서비스 핵심 정체성 (가성비 중심) |

### 사용자 가중치 정규화

$$
w_i = \frac{s_i}{\sum_{j \in \{Q,E,S,P\}} s_j}
$$

> $s_i$: 사용자 슬라이더 입력 (1~5점)
> $w_i$: 정규화된 개인 선호 (합 = 1)
